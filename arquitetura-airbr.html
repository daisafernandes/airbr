Visão completa das integrações, camadas e fluxo de dados

① Fontes de Dados
🇧🇷 APIs Brasileiras Oficiais
🔥
INPE / BDQueimadas
Focos de incêndio via satélite, atualização a cada 3h. Cobre todo o Brasil e América Latina.
gratuito BR oficial
🏭
CETESB
Rede de estações em São Paulo. AQI hora a hora, PM2.5, PM10, O₃, NO₂. Sistema QUALAR.
gratuito BR oficial
📊
IEMA Nacional
Plataforma com dados históricos de 11 estados. Usado pela OMS. 478 registros, 82 localidades.
gratuito BR oficial
🌿
IAT / Paraná
Monitoramento oficial da Região Metropolitana de Curitiba. Índice IQA diário e boletins.
gratuito BR oficial
🌍 APIs Globais (cobertura via satélite)
🌐
IQAir API
AQI em tempo real, PM2.5, previsão de 3 dias. Melhor cobertura de cidades brasileiras.
free + pago
📡
AQICN API
+500 estações no Brasil. Dados de estações locais em tempo real. Token gratuito.
gratuito
☁️
OpenWeatherMap
Air Pollution API cobre todo o Brasil por coordenadas. 1.000 requisições/dia grátis.
gratuito
🛰️
NASA FIRMS
Dados de queimadas e incêndios por satélite MODIS e VIIRS. Complementa o INPE.
gratuito
🇪🇺
Copernicus / CAMS
Modelo atmosférico europeu. Previsão de qualidade do ar para toda a América do Sul.
gratuito
🌡️
Open-Meteo
Clima + qualidade do ar + índice UV combinados. API aberta, sem necessidade de chave de acesso.
gratuito
🌿 Dados Ambientais Complementares
🌳
PRODES / INPE
Monitoramento de desmatamento em tempo real. Correlacionar áreas desmatadas com qualidade do ar e focos de queimada.
gratuito BR oficial
🌼
Ambee Pollen API
Índice de pólen e alérgenos por cidade. Essencial para usuários asmáticos e com rinite. Cobre o Brasil.
free + pago
☀️
Índice UV (Open-Meteo)
Combinado com AQI forma um "índice de segurança ao ar livre". Alta exposição UV + poluição = risco duplicado.
gratuito
🏥 Saúde Pública e Contexto Social
🏥
DATASUS / SUS
Internações por doenças respiratórias por município. Cruzar com picos de poluição para mostrar impacto real na saúde.
gratuito BR oficial
👥
IBGE API
Dados populacionais por município: idosos, crianças, densidade. Mostrar quantas pessoas vivem em risco em cada cidade.
gratuito BR oficial
💡 Estratégia de cobertura: Onde há estação oficial (11 estados) → usar CETESB/IEMA. Onde não há → satélite via IQAir + OpenWeatherMap. Queimadas + desmatamento → INPE/PRODES sempre. Saúde pública → DATASUS + IBGE.
🏆 Diferencial único no Brasil: Cruzar focos de queimada (INPE) + direção do vento (Open-Meteo) + AQI (satélite) + internações (DATASUS) = "A fumaça do Mato Grosso está afetando a qualidade do ar em São Paulo hoje e causou X internações na semana passada."
INGESTÃO DE DADOS
② Camada de Ingestão (Backend — Node.js ou Python)
⏱️
Scheduler / Cron
Jobs agendados que buscam dados nas APIs externas em intervalos definidos: a cada 1h para AQI, a cada 3h para queimadas.
🔄
Normalização
Converte os diferentes formatos (JSON, CSV, XML) de cada API para um esquema único e padronizado antes de salvar.
🗝️
Gerenciador de Chaves
Armazena todas as API keys com segurança em variáveis de ambiente. Nunca expostas ao frontend.
Fluxo de normalização

CETESB XML

INPE CSV

DATASUS JSON

IBGE JSON

ARMAZENAMENTO E CACHE
③ Armazenamento e Cache
💾 Banco de Dados
🐘
PostgreSQL + PostGIS
Dados históricos de AQI por cidade. PostGIS permite consultas geoespaciais (focos próximos, raio de poluição).
🍃
MongoDB (alternativa)
Mais simples de escalar. Ideal para protótipos. Armazena documentos JSON diretamente.
⚡ Cache em Memória
🔴
Redis
Cache dos dados mais recentes por cidade. TTL de 1h para AQI. Evita exceder limites das APIs gratuitas.
📁
JSON em disco (protótipo)
Para começar rápido: salvar snapshots JSON localmente. Simples e sem dependências extras.
API INTERNA
④ API REST Interna (o que o frontend consome)
🗺️
GET /cities
Lista todas as cidades monitoradas com AQI atual, coordenadas e fonte dos dados.
📍
GET /city/:id
Dados completos de uma cidade: AQI, PM2.5, PM10, CO, NO₂, O₃, histórico 7 dias, alertas.
🔥
GET /fires
Focos de queimada ativos por região/estado. Retorna coordenadas, intensidade e satélite de origem.
🏆
GET /ranking
Ranking das cidades mais e menos poluídas do dia/semana. Filtrável por região e estado.
📈
GET /history/:city
Histórico de AQI por período (7d, 30d, 1 ano). Base para gráficos e tendências.
📍
GET /nearby
Cidades monitoradas próximas a uma coordenada (geolocalização do usuário).
🔔
POST /alerts
Cadastro de alertas: usuário escolhe cidade e nível de AQI para receber notificação.
🔍
GET /search
Busca por nome de cidade com autocomplete. Retorna lista com estado, região e AQI atual.
🏥
GET /health/:city
Dados de saúde pública: internações respiratórias, grupos de risco por faixa etária e correlação com AQI histórico.
🌳
GET /deforestation
Áreas desmatadas recentes do PRODES/INPE por estado. Correlacionadas com focos de queimada e AQI regional.
💨
GET /wind-smoke/:city
Cruza direção do vento (Open-Meteo) + focos do INPE para indicar de onde vem a fumaça que afeta a cidade.
☀️
GET /outdoor-safety/:city
Índice composto: AQI + UV + pólen + temperatura. Retorna nível de segurança para atividades ao ar livre.
🌍
GET /oms-compliance
Cidades acima/abaixo do limite de PM2.5 recomendado pela OMS (5 µg/m³). Ranking de conformidade.
FRONTEND
⑤ Frontend (o que o usuário vê)
🗺️
Mapa Interativo do Brasil
Cidades coloridas por nível de AQI. Clicável por estado ou cidade. Camada de queimadas sobrepostas.
🔍
Busca com Autocomplete
Campo de busca com sugestões em tempo real. Resultado instantâneo ao selecionar cidade.
📍
Geolocalização
Botão "usar minha localização". Detecta cidade mais próxima e carrega dados automaticamente.
📊
Dashboard da Cidade
AQI atual com gauge, poluentes individuais, gráfico histórico 7 dias, previsão e fonte dos dados.
⚠️
Alertas de Saúde
Recomendações dinâmicas por faixa de AQI: grupos sensíveis, crianças, idosos, atividade física.
🏆
Ranking Nacional
As 10 cidades mais poluídas e mais limpas do dia. Atualizável por região ou estado.
🔥
Mapa de Queimadas
Focos ativos do INPE sobrepostos no mapa. Indicador de risco para qualidade do ar nas regiões afetadas.
🔔
Notificações
Push notification ou e-mail quando o AQI da cidade favorita ultrapassa limite definido pelo usuário.
🏥
Painel de Saúde Pública
Internações respiratórias por município cruzadas com histórico de AQI. Mostra o impacto real da poluição em números.
💨
De onde vem a fumaça?
Visualização do trajeto da fumaça: direção do vento + focos INPE. Ex: "fumaça do MT está afetando SP hoje".
☀️
Índice "Seguro ao Ar Livre"
Score combinado de AQI + UV + pólen + temperatura. Verde = sair é seguro. Vermelho = fique em casa.
🌍
Compliance com OMS
Quais cidades atendem (ou não) os limites de PM2.5 da OMS. Ranking de conformidade nacional e comparativo global.
🌳
Mapa de Desmatamento
Camada do PRODES sobreposta ao mapa. Correlaciona áreas desmatadas com aumento de queimadas e piora do AQI.
